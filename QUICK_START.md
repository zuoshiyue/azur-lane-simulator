# 🎮 碧蓝航线阵容模拟器 - 快速使用指南

## 🌐 立即访问

**在线版本**: https://zuoshiyue.github.io/azur-lane-simulator

## 📱 三个主要页面

### 1️⃣ 阵容模拟
- **功能**: 拖拽角色组建舰队
- **位置**: 点击顶部"阵容模拟"按钮
- **操作**:
  - 从左侧角色池拖拽角色到右侧槽位
  - 先锋舰队（前 3 个）+ 主力舰队（后 3 个）
  - 点击"智能推荐"获取推荐阵容
  - 点击"导出阵容"保存为 JSON

### 2️⃣ 角色数据库
- **功能**: 查询所有角色信息
- **位置**: 点击顶部"角色数据库"按钮
- **操作**:
  - 搜索角色名、类型、阵营
  - 筛选舰种和阵营
  - 点击角色卡片查看详情
  - 切换网格/列表视图

### 3️⃣ 角色池管理 ⭐ 新增
- **功能**: 完整 CRUD 管理角色池
- **位置**: 点击顶部"角色池管理"按钮
- **操作**: 见下方详细说明

---

## 🎯 角色池管理功能

### ➕ 添加角色
1. 点击绿色 **"添加角色"** 按钮
2. 填写表单：
   - **必填**: 英文名、中文名、舰种、阵营
   - **选填**: 图片 URL、属性、技能、装备
3. 点击 **"保存"**

### ✏️ 编辑角色
1. 悬停要编辑的角色卡片
2. 点击右上角 **✏️ 编辑图标**
3. 修改信息
4. 点击 **"保存"**

### 🗑️ 删除角色
**单个删除**:
1. 悬停角色卡片
2. 点击 **🗑️ 删除图标**
3. 确认删除

**批量删除**:
1. 勾选多个角色（左上角复选框）
2. 点击红色 **"删除选中"** 按钮
3. 确认删除

### 🔍 搜索和筛选
- **搜索框**: 输入角色名（中文/英文）、舰种、阵营
- **舰种筛选**: 选择特定舰种或"全部"
- **阵营筛选**: 选择特定阵营或"全部"

### 📊 查看统计
顶部统计面板显示：
- 总角色数（蓝色）
- 已拥有（绿色）
- SSR 6★（黄色）
- SR 5★（紫色）
- R 4★（蓝色）
- N ≤3★（灰色）
- 筛选结果（粉色）

### 💾 导入导出
1. 点击 **"导入/导出"** 按钮
2. **导出**: 点击"导出 JSON 文件"
3. **导入**: 
   - 方式 1: 点击"选择 JSON 文件"
   - 方式 2: 粘贴 JSON 文本 → 点击"导入文本"

## 💡 使用技巧

### 快速上手
1. **新手**: 先去"角色数据库"浏览现有角色
2. **管理**: 到"角色池管理"添加/编辑角色
3. **组队**: 在"阵容模拟"拖拽角色组建舰队

### 数据备份
- 定期在"角色池管理"点击"导入/导出" → "导出 JSON 文件"
- 清除浏览器缓存会丢失数据！

### 批量操作
- 使用复选框选择多个角色
- 一次性删除不需要的角色
- 提高效率

### 搜索技巧
- 输入"企业"或"Enterprise"都能搜索到
- 输入"航母"筛选所有航母角色
- 输入"白鹰"筛选白鹰阵营角色

## 🔧 常见问题

### Q: 添加的角色在哪里？
A: 添加后自动保存到角色池，在"角色池管理"和"阵容模拟"的角色列表中都能看到。

### Q: 如何修改角色属性？
A: 在"角色池管理"页面，悬停角色卡片 → 点击编辑图标 → 修改 → 保存。

### Q: 删除角色后阵容会怎样？
A: 如果角色在阵容中，系统会提示你并自动从阵容中移除该角色。

### Q: 数据会同步吗？
A: 数据保存在浏览器 localStorage，仅在当前设备有效。更换设备需要导出导入。

### Q: 可以导入其他人的角色池吗？
A: 可以！让对方导出 JSON 文件，你导入即可。系统会自动合并。
