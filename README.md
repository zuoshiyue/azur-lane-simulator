## 🎮 使用指南

### 阵容模拟
1. 点击导航栏"阵容模拟"
2. 从左侧角色池拖拽角色到阵容槽位
3. 查看实时战力计算
4. 点击"智能推荐"获取推荐阵容
5. 使用底部的"保存/加载"功能管理阵容
6. 切换到另一个海上阵容标签页对比不同配置

### 编队模式切换
1. 在阵容模拟器页面切换编队模式
2. **先锋编队** - 3 槽位（驱逐/轻巡/重巡/超巡/运输）
3. **主力编队** - 3 槽位（战列/战巡/航母/轻母/潜艇/维修）
4. **潜艇编队** - 6 槽位（仅潜艇）

### 角色池管理
1. 点击导航栏"角色池管理"
2. 使用搜索框输入角色名（支持中文、别称搜索）
3. 使用筛选器按舰种/阵营过滤
4. 点击"设为已拥有"按钮标记拥有的角色
5. 点击"批量添加已拥有"快速添加筛选结果
6. 点击"已拥有"按钮只显示已拥有的角色
7. 点击"导出/导入数据"按钮可以一键导出和导入已拥有角色数据
8. 悬停角色卡片显示编辑/删除按钮
9. 使用复选框批量删除

### 阵容管理
1. 在阵容模拟器中使用保存/加载功能
2. 为阵容命名方便识别
3. 使用对比功能比较不同阵容
4. 一键导出阵容分享给他人

### 主题切换
1. 点击导航栏中的主题切换按钮
2. 在浅色、深色和自动模式间切换
3. 设置会持久化保存

### 设置拥有角色
1. 进入"角色池管理"
2. 点击角色卡片的"设为已拥有"按钮
3. 已拥有角色显示绿色标记
4. 智能推荐会优先使用已拥有角色

### OCR舰舱识别（v3.4）
1. 在阵容模拟器界面点击"识别截图"按钮
2. 上传您的舰舱/仓库截图（支持JPG/PNG等格式）
3. 系统将自动识别截图中的角色名称
4. 查看识别结果和置信度评分
5. 点击"导入到角色库"将识别出的角色添加到收藏
6. 可以选择重新上传或处理现有截图

## 📝 开发说明

### 数据格式
```typescript
interface Character {
  id: string;
  name: string;        // 英文名
  nameCn: string;      // 中文名（默认显示）
  rarity: number;      // 稀有度 1-6
  type: string;        // 舰种
  faction: string;     // 阵营
  stats: Stats;        // 属性
  skills: Skill[];     // 技能
  equipment: EquipmentSlot[];  // 装备
}
```

### 推荐算法说明
推荐系统基于以下维度综合计算：

- **角色稀有度** - UR > SSR > SR > R > N（权重：85/80/60/40/20）
- **舰种强度系数** - 航母(1.15) > 轻母(1.12) > 战巡(1.05) > 战列(1.10)等
- **基础属性** - HP、火力、鱼雷、航空、装填、对空、索敌等
- **舰种定位** - 前排（先锋）和后排（主力）的角色分配
- **阵营协同** - 同阵营加成效果（最高可达15%）
- **位置权重** - 后排角色通常获得10%额外权重
- **特殊优化** - 不同模式（如BOSS队、道中队、1拖5等）的针对性策略

**注意**：算法仍在持续优化中，如果您发现某些强力角色（如独角兽）未被优先推荐，请向我们反馈！
