## 🔧 使用说明

### 添加角色到角色池
1. 打开阵容模拟器页面
2. 点击绿色"添加角色"按钮
3. 在搜索框中输入角色名或使用筛选器
4. 点击角色卡片右上角的"+"按钮
5. 角色已添加到角色池，可在左侧查看

### 组建阵容
1. 从左侧角色池拖拽角色到右侧阵容槽位
2. 前 3 个位置为先锋舰队（驱逐、轻巡、重巡）
3. 后 3 个位置为主力舰队（战列、航母等）
4. 点击"智能推荐"获取系统推荐阵容
5. 点击"导出阵容"保存为 JSON 文件

### 搜索功能
- 支持中文名搜索（如"企业"）
- 支持英文名搜索（如"Enterprise"）
- 支持舰种搜索（如"航母"）
- 支持阵营搜索（如"白鹰"）
