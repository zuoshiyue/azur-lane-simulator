# 碧蓝航线阵容模拟器 - 智能编队推荐功能

## 📋 功能概述

智能编队推荐系统根据用户拥有的角色情况进行个性化推荐，综合考虑舰种搭配、阵营协同、强度评级和阵容平衡性。

## ✨ 核心功能

### 1. 用户拥有系统

**功能特性：**
- ✅ 每个角色添加"拥有状态"标记
- ✅ 角色池管理界面添加"设为已拥有/取消拥有"按钮
- ✅ 已拥有角色列表保存到 localStorage
- ✅ 视觉区分：已拥有角色显示绿色边框和勾选标记

**使用方法：**
1. 进入"角色池管理"页面
2. 在任意角色卡片上点击"设为已拥有"按钮
3. 已拥有角色会显示绿色标记
4. 再次点击可取消拥有状态

### 2. 智能推荐算法

**推荐维度：**

#### a) 用户拥有的角色（优先推荐）
- 只从用户已拥有的角色中选择
- 确保推荐阵容可用

#### b) 舰种搭配
- **先锋舰队（前排）**：驱逐、轻巡、重巡、超巡
- **主力舰队（后排）**：战列、战巡、航母、轻母
- **支援角色**：维修、运输（可灵活分配）
- 理想比例：3 前排 + 3 后排

#### c) 阵营协同
- 同阵营角色获得额外加成
- 阵营加成系数：
  - 白鹰：1.15
  - 皇家：1.12
  - 重樱/铁血：1.10
  - 东煌/北方联合/撒丁帝国：1.08
  - 自由鸢尾/维希教廷：1.05

#### d) 强度评级
基于以下因素计算角色强度分：
- 基础属性（HP、火力、鱼雷、航空、装填等）
- 稀有度权重（SSR > SR > R > N）
- 舰种系数（航母/战列 > 其他）
- 技能数量加成

#### e) 阵容平衡性
- 前后排比例检查
- 舰种多样性评估
- 自动识别并提示阵容缺陷

### 3. 推荐 UI

**界面元素：**
- 🎯 "智能推荐"按钮（阵容模拟器页面）
- 📊 推荐结果面板（模态框形式）
- 👥 显示推荐阵容（6 人：先锋 3 + 主力 3）
- ⚡ 一键应用到当前阵容
- 📝 详细推荐理由展示

**推荐理由包含：**
- 推荐模式标识
- 阵营协同加成百分比
- 阵容平衡性评价
- 综合战力评分
- 潜在问题提示

### 4. 推荐模式

#### 🏆 最强阵容
- 基于综合强度的最优解
- 优先选择高评分角色
- 适合追求极限战力的玩家

#### 🎌 阵营队
- 单一阵营组成（如纯白鹰队）
- 最大化阵营协同加成
- 用户可选择具体阵营

#### 🌟 新手友好
- 优先推荐容易获取的角色
- 排除高稀有度（SSR）角色
- 适合新手玩家参考

#### ⚙️ 自定义
- 用户可指定特定条件
- 灵活调整推荐参数
- 满足特殊需求

### 5. 数据持久化

**本地存储内容：**
- ✅ 拥有列表（localStorage: `azur_lane_owned_characters`）
- ✅ 阵容数据（localStorage: `azur_lane_fleets`）
- ✅ 自动保存，无需手动操作

**存储格式：**
```json
{
  "ownedCharacterIds": ["char_001", "char_002", ...],
  "fleets": [...]
}
```


## 📊 算法详情

### 角色强度计算公式

```typescript
基础分 = HP×0.5 + 火力×1.2 + 鱼雷×1.3 + 航空×1.5 + 装填×0.8 + 防空×0.6 + 索敌×0.5
稀有度加成 = {SSR:80, SR:60, R:40, N:20}
舰种系数 = {航母:1.15, 战列:1.10, 超巡:1.00, ...}
技能加成 = 1 + 技能数×0.1

最终评分 = (基础分 + 稀有度加成) × 舰种系数 × 技能加成
```

### 阵容评分计算

```typescript
基础战力 = Σ(角色评分 × 位置系数)
阵营加成 = 同阵营角色比例 × 阵营系数
平衡系数 = 1.0 - (缺陷数量 × 0.05)

最终评分 = 基础战力 × 阵营加成 × 平衡系数
```

## 🎯 功能亮点

1. **个性化推荐** - 基于实际拥有情况，不推荐未获得角色
2. **多维度评估** - 综合考虑强度、协同、平衡性
3. **多种模式** - 满足不同玩家需求（强度党/收集党/新手）
4. **透明理由** - 清晰展示推荐依据和评分来源
5. **一键应用** - 快速将推荐阵容应用到模拟器
6. **本地存储** - 数据安全，无需登录

## 🔧 技术实现

### 文件结构

```
src/
├── components/
│   ├── FleetRecommendationPanel.tsx  # 推荐面板组件
│   ├── CharacterCard.tsx             # 角色卡片（支持拥有标记）
│   ├── CharacterPoolManager.tsx      # 角色池管理
│   └── FleetSimulator.tsx            # 阵容模拟器（集成推荐）
├── utils/
│   └── recommender.ts                # 推荐算法核心
├── data/
│   └── dataManager.ts                # 数据管理（拥有状态）
└── types.ts                          # 类型定义
```

### 核心 API

```typescript
// 推荐阵容
recommendFleet(ownedCharacters, mode, options)

// 计算角色强度
calculateCharacterPower(character)

// 计算阵容战力
calculateFleetPower(fleet)

// 获取已拥有角色
dataManager.getOwnedCharacterIds()

// 设置拥有状态
dataManager.addCharacter(character)
dataManager.deleteCharacter(characterId)
```
