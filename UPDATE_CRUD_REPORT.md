## ✨ 新增功能

### 1. 增（Create）✅

#### 添加新角色
- **入口**: 角色池管理页面 → "添加角色"按钮
- **表单字段**:
  - 基本信息：英文名、中文名、稀有度（1-6 星）、舰种、阵营、图片 URL
  - 基础属性：耐久、火力、鱼雷、航空、装填、装甲、速度、幸运、防空、探测
  - 技能系统：可添加多个技能，支持主动/被动类型、冷却时间
  - 装备槽：可自定义装备槽类型和效率
- **验证**: 必填字段验证、数值范围检查
- **保存**: 自动生成唯一 ID，保存到本地存储

#### 批量导入
- 支持 JSON 文件导入
- 支持粘贴 JSON 文本导入
- 自动合并：已有角色更新，新角色添加
- 导入提示：显示导入成功数量

---

### 2. 删（Delete）✅

#### 单个删除
- 悬停角色卡片显示删除按钮
- 删除前确认弹窗
- **智能检测**: 如果角色在阵容中使用，提示用户并自动从阵容中移除

#### 批量删除
- 支持多选角色（复选框）
- 显示选中数量
- 批量删除确认
- 一次性删除多个角色

#### 删除保护
- 删除确认弹窗防止误操作
- 阵容使用检测
- 二次确认提示

---

### 3. 改（Update）✅

#### 编辑角色
- 悬停角色卡片显示编辑按钮
- 打开完整表单，预填充现有数据
- 修改所有字段（基本信息、属性、技能、装备）
- 实时更新到本地存储
- 表单验证同添加功能

#### 实时更新
- 保存后立即刷新列表
- 阵容中的角色数据同步更新
- 无需刷新页面

---

### 4. 查（Read）✅

#### 搜索功能
- 支持中文名搜索
- 支持英文名搜索
- 支持舰种搜索
- 支持阵营搜索
- 实时过滤结果

#### 筛选功能
- 舰种筛选：12 种舰船类型
- 阵营筛选：白鹰、皇家、铁血、重樱等
- 组合筛选：搜索 + 舰种 + 阵营

#### 统计面板
7 个统计卡片显示：
- **总角色数**: 数据库中所有角色数量
- **已拥有**: 当前角色池中的角色数
- **SSR (6★)**: 6 星角色数量（黄色）
- **SR (5★)**: 5 星角色数量（紫色）
- **R (4★)**: 4 星角色数量（蓝色）
- **N (≤3★)**: 3 星及以下数量（灰色）
- **筛选结果**: 当前筛选条件下的数量（粉色）

#### 视图切换
- **网格视图**: 卡片式展示，适合浏览
- **列表视图**: 紧凑列表，适合管理

## 📖 使用指南

### 添加角色
1. 点击顶部导航栏"角色池管理"
2. 点击绿色"添加角色"按钮
3. 填写完整信息（带*为必填）
4. 点击"保存"

### 编辑角色
1. 在角色池中找到要编辑的角色
2. 悬停卡片，点击右上角 ✏️ 编辑图标
3. 修改信息
4. 点击"保存"

### 删除角色
**单个删除**:
1. 悬停角色卡片
2. 点击 🗑️ 删除图标
3. 确认删除

**批量删除**:
1. 勾选要删除的角色（卡片左上角复选框）
2. 点击红色"删除选中"按钮
3. 确认删除

### 导入数据
**文件导入**:
1. 点击"导入/导出"按钮
2. 点击"选择 JSON 文件"
3. 选择之前导出的 JSON 文件
4. 自动导入并提示成功数量

**文本导入**:
1. 点击"导入/导出"按钮
2. 在文本框粘贴 JSON
3. 点击"导入文本"

### 导出数据
1. 点击"导入/导出"按钮
2. 点击"导出 JSON 文件"
3. 自动下载 JSON 文件
