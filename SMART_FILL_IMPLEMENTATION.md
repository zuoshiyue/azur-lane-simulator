# 碧蓝航线阵容模拟器 - 智能角色添加功能

## 📋 实现概述

实现了"输入名字自动补充数据"的智能添加功能，用户可以通过输入角色名称自动填充表单数据。

## 🎯 功能特性

### 1. 智能填充 UI
- **入口**: 在角色添加/编辑表单右上角添加"智能填充"按钮（紫色，带 Sparkles 图标）
- **交互**: 点击按钮展开智能填充面板
- **输入**: 支持输入角色名称，带实时搜索建议
- **状态**: 显示加载状态、成功/错误消息

### 2. 数据源方案

#### 方案一：本地模板库（当前主要方案）✅
- **位置**: `src/data/characterTemplates.ts`
- **支持角色**: 10 个常见角色模板
  - 企业（白鹰，航母，5 星）
  - 贝尔法斯特（皇家，轻巡，5 星）
  - 赤城（重樱，航母，5 星）
  - 加贺（重樱，航母，5 星）
  - 大和（重樱，战列，6 星）
  - 武藏（重樱，战列，6 星）
  - 俾斯麦（铁血，战列，5 星）
  - 提尔比茨（铁血，战列，5 星）
  - 胡德（皇家，战巡，5 星）
  - 厌战（皇家，战列，5 星）

- **功能**:
  - 精确匹配搜索
  - 模糊搜索支持
  - 完整的属性、技能、装备数据

#### 方案二：B 站 Wiki 爬虫（预留接口）⏸️

- **未来扩展**: 可通过后端代理实现 Wiki 数据获取

## 🎨 UI 交互流程

1. 用户点击"添加角色"按钮
2. 在表单右上角点击"智能填充"按钮（紫色）
3. 输入角色名称（如"企业"）
4. 系统自动搜索并显示建议
5. 点击"查询"或直接选择建议
6. 显示加载状态
7. 成功填充表单数据
8. 用户可手动修改后保存
